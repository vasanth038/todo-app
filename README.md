# Todo App 📝

A simple and responsive Todo application built using **React** that allows users to manage their daily tasks efficiently.

## 🚀 Live Demo
[Click here to view the app](https://todo-app-three-kappa-32.vercel.app/)

---

## ✨ Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using **localStorage**
- Clean and minimal design

---

## 🛠️ Tech Stack

- **Frontend:** React, JavaScript, HTML, CSS  
- **State Management:** React Hooks (`useState`, `useEffect`)  
- **Storage:** Browser LocalStorage  
- **Deployment:** Vercel
